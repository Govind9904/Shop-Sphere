# ShopSphere

**ShopSphere** is a professional, full-stack e-commerce platform built with **React.js**, **Node.js**, and **MongoDB**.  
It is designed to be scalable, modern, and portfolio-ready, featuring a responsive UI, JWT authentication, admin dashboard, and payment integration with **Razorpay / Stripe** (test mode).

## üî• Features

### User Features
- User registration & login with JWT authentication
- Email verification & password reset
- Product listing with search, filters, and pagination
- Product details with multiple images
- Cart & Wishlist management
- Secure checkout with Razorpay / Stripe
- Order history and status tracking
- Responsive design (mobile + desktop)

### Admin Features
- Admin login with role-based access control
- Product management (add/update/delete products)
- Category management
- Order management with status updates
- Dashboard analytics (sales, top products, active users)

### Extra Professional Features
- Password encryption with bcrypt
- File uploads for product images using Multer
- Email notifications using Nodemailer
- Caching for product catalog with Redis
- Security features: rate limiting, helmet, input validation
- Clean and modern UI with React + TailwindCSS / MUI
